01 C · Interface terminal · Algorithme

Boggle Game

Jeu de lettres complet en C avec interface ncurses. Grille 4×4 générée aléatoirement, dictionnaire trié par recherche dichotomique, validation des mots en temps réel. Architecture en 12 fichiers C séparés par responsabilité.

Ce que j'ai fait

Structure de données dictionnaire en tableau trié + recherche binaire O(log n). Gestion des entrées clavier via ncurses. Séparation claire des modules.

Limites honnêtes

Pas de score persistant. Pas d'IA adverse. Site statique uniquement.

Ce que j'ajouterais

Mode multijoueur réseau avec sockets. Sauvegarde en base de données.

03 Algorithmique · C · Python · Analyse

Arbres Binaires
de Recherche

Projet allant au-delà de l'implémentation : vérification et génération d'ABR, analyse de complexité formelle, benchmarking sur 10 000 arbres générés, visualisations matplotlib et rapport complet avec preuves algorithmiques.

Ce que j'ai fait

Implémentation C de la vérification ABR en O(n). Scripts Python de génération et benchmarking. Figures matplotlib automatisées.

Ce qui m'a rendu fier

Passer de « ça marche » à « je prouve pourquoi ça marche en O(n) et pas O(n²) ».

vérification ABR — O(n)
bool is_bst(Node *n, int min, int max) {
    if (!n) return true;
    return n->val > min && n->val < max
        && is_bst(n->left,  min,    n->val)
        && is_bst(n->right, n->val, max);
}
